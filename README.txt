EquiParental — Site statique (HTML/CSS)
======================================

Contenu :
- index.html (Accueil)
- about.html (À propos)
- services.html (Services)
- contact.html (Contact + formulaire)
- merci.html (Page de confirmation)
- styles.css (Feuille de style)
- assets/logo.png (Logo fourni)
- README.txt (vous lisez ce fichier)

Formulaire de contact :
- Le formulaire est configuré pour utiliser <https://formsubmit.co> afin d'envoyer directement les messages à equiparental@outlook.fr sans configuration serveur.
- Si vous préférez utiliser Formspree (formspree.io), créez un formulaire sur Formspree, récupérez votre endpoint (ex: https://formspree.io/f/xxxxx) et remplacez l'attribut action du formulaire dans contact.html par votre endpoint.

Hébergement :
- Vous pouvez publier ce dossier sur GitHub Pages (push du repository et activer Pages) ou sur Netlify / Vercel (drag-and-drop du dossier ou connect au repo).
- Exemple rapide pour GitHub Pages :
  1. Créez un nouveau dépôt GitHub.
  2. Ajoutez tous les fichiers et poussez.
  3. Dans les paramètres du dépôt, activez GitHub Pages (branche main / dossier root).
  4. Le site sera disponible à https://<votre-nom>.github.io/<repo>/

Modification rapide :
- Ouvrez styles.css pour ajuster les couleurs ou la typo.
- Remplacez assets/logo.png par votre logo si vous souhaitez une version différente.

Besoin d'aide :
- Si vous voulez que je déploie le site sur GitHub Pages (je peux vous fournir les étapes), ou que j'adapte le design, dites-moi exactement ce que vous souhaitez changer.

Bonne publication !
